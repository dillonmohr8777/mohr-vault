# SOP: Facebook Ads Weekly Optimization

## When to Use
Every week for each active Meta ad account.

## Step 1: Performance Review
- Pull last 7 days data
- Compare to previous 7 days
- Check: spend, CPA, CTR, CPM, frequency, ROAS

## Step 2: Identify Issues
- [ ] Any campaign spending with 0 conversions?
- [ ] Any ad set with frequency above 3?
- [ ] Any ad with CTR below 0.8%?
- [ ] Any CPM spikes?
- [ ] Any disapproved ads?

## Step 3: Take Action
- Pause ads with CTR below 0.5% after 1,000+ impressions
- Pause audiences with CPA 2x above target after sufficient spend
- Increase budget on ad sets with CPA below target (max 20% increase)
- Launch queued tests if current tests have concluded
- Refresh creative if frequency above 3

## Step 4: Document
- Update Reporting Log for each client
- Log any budget changes in Budget Shift Log
- Add learnings to client's Agent Memory
- Update Testing Queue

## Step 5: Notify
- Send Slack summary if significant changes made
- Flag anything needing client approval
