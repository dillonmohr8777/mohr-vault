# SOP: Facebook Ads Creative Testing

## When to Use
When testing new creative concepts, copy, or offers.

## Testing Rules
1. Only test ONE variable at a time
2. Keep everything else identical
3. Let tests run minimum 3-5 days
4. Need minimum 1,000 impressions per variation before calling it
5. Statistical significance matters - don't kill too early

## Test Types

### Image/Video Test
- Same copy, same audience, different creative
- Run 2-3 variations
- Winner = lowest CPA or highest CTR depending on goal

### Copy Test
- Same creative, same audience, different primary text
- Test hooks, angles, length

### Audience Test
- Same creative, same copy, different audiences
- Compare custom vs lookalike vs interest vs broad

### Offer Test
- Same audience, different value propositions
- e.g., "Free consultation" vs "50% off first month"

## Documentation
After each test:
1. Record in Testing Queue (completed section)
2. Update client's Creative Angles with winning elements
3. Add learning to Agent Memory
4. Apply winner to other campaigns if applicable
