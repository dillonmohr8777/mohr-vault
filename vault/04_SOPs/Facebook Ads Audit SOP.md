# SOP: Facebook Ads Account Audit

## When to Use
When onboarding a new client or doing a periodic deep review.

## Step 1: Account Health
- [ ] Account status (active, restricted, disabled?)
- [ ] Payment method valid?
- [ ] Business verification complete?
- [ ] Any policy violations or warnings?

## Step 2: Pixel & Tracking
- [ ] Pixel installed?
- [ ] Pixel firing correctly?
- [ ] Standard events configured?
- [ ] Conversions API set up?
- [ ] Domain verified?

## Step 3: Campaign Structure
- [ ] Are campaigns organized logically?
- [ ] Is naming convention consistent?
- [ ] Are objectives appropriate for goals?
- [ ] Is budget allocation sensible?

## Step 4: Audience Review
- [ ] Are custom audiences up to date?
- [ ] Are lookalike audiences created?
- [ ] Is there audience overlap between ad sets?
- [ ] Are exclusions set properly?

## Step 5: Creative Review
- [ ] Is creative fresh or stale?
- [ ] Are ad formats varied (image, video, carousel)?
- [ ] Is copy compelling and on-brand?
- [ ] Are CTAs clear and appropriate?

## Step 6: Performance Analysis
- [ ] What's the average CPA? Is it acceptable?
- [ ] Which campaigns/ad sets/ads are winners?
- [ ] Where is budget being wasted?
- [ ] What's the frequency? (ad fatigue?)

## Step 7: Document & Recommend
- Fill out client's Facebook Ads Account Notes
- Update Agent Memory with key findings
- Create prioritized action list
- Send audit summary via Slack
