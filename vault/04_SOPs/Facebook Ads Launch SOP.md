# SOP: Facebook Ads Campaign Launch

## When to Use
When launching a new campaign or ad set for any client.

## Pre-Launch Checklist
- [ ] Pixel installed and firing correctly
- [ ] Conversion events set up (Lead, Purchase, etc.)
- [ ] UTM parameters configured
- [ ] Landing page tested and loading fast
- [ ] Ad copy reviewed (no policy violations)
- [ ] Creative assets sized correctly (1080x1080, 1080x1920, 1200x628)
- [ ] Audience defined and saved
- [ ] Budget confirmed with client
- [ ] Special Ad Category set if applicable (Housing, Credit, Employment)
- [ ] Exclusions set (existing customers, converters, etc.)

## Campaign Structure
1. Choose objective (Leads, Sales, Traffic, Awareness)
2. Set campaign budget optimization (CBO) or ad set budget
3. Name convention: [Client] - [Objective] - [Audience] - [Date]

## Ad Set Setup
1. Define audience (interests, lookalike, custom, broad)
2. Set placements (Advantage+ or manual)
3. Set schedule and budget
4. Set optimization event
5. Name convention: [Audience Type] - [Targeting Detail]

## Ad Setup
1. Select format (single image, carousel, video)
2. Add primary text, headline, description
3. Add CTA button
4. Add destination URL with UTMs
5. Name convention: [Angle] - [Creative Type] - [Version]

## Post-Launch (First 48 Hours)
- [ ] Confirm ads are approved and delivering
- [ ] Check pixel is firing on landing page
- [ ] Monitor for unusual CPM or delivery issues
- [ ] Do NOT touch anything for 48-72 hours (let it learn)

## Notes
- Never edit ads during learning phase
- If an ad gets rejected, duplicate and fix rather than editing
