# SOP: Google Ads Weekly Optimization

## When to Use
Every week for each active Google Ads account.

## Step 1: Performance Review
- Pull last 7 days; compare to previous 7 days.
- Check: spend, conversions, CPA vs target, conv. rate, CTR, impression share.
- [ ] CPA within target? Flag any campaign 2x over target.
- [ ] Any campaign spending with 0 conversions?

## Step 2: Budget & Pacing
- [ ] Spend pacing to monthly budget (not under/over by >10%)
- [ ] "Limited by budget" flag on any campaign? Note for budget conversation.
- [ ] Shift budget toward campaigns hitting CPA target (max ~20% move at once)

## Step 3: Search Terms & Negatives
- [ ] Run Search Terms Review SOP (separate doc)
- [ ] Add new negatives from wasted spend
- [ ] Promote strong converting terms to their own exact-match keyword

## Step 4: Bid Strategy Health
- [ ] Learning status = not stuck in "Learning"
- [ ] If on tCPA/tROAS: is actual CPA tracking to target? Adjust target by max 15-20% per change.
- [ ] Enough conversion volume to justify current strategy? If volume dropped, reassess.

## Step 5: Keyword & Ad Triage
- [ ] Low Quality Score keywords (below 5) — review relevance, ad copy, landing page; pause chronic low performers
- [ ] Ad Strength below "Good" — add/refresh headlines and descriptions
- [ ] Refresh RSA assets on stale or low-CTR ads
- [ ] Sitelink/callout/asset performance — pause weak assets, add new ones

## Step 6: Bid Adjustments
- [ ] Geo report — pull location performance; bid down/exclude poor areas, bid up strong ones
- [ ] Device report — adjust mobile/desktop/tablet if CPA differs materially

## Step 7: Impression Share Audit
- [ ] Search Lost IS (budget) high? → budget conversation
- [ ] Search Lost IS (rank) high? → Quality Score / bid / ad relevance issue

## Step 8: Wasted Spend Audit
- [ ] Keywords with spend and 0 conversions over threshold (e.g. 1.5x target CPA in cost) — pause or fix
- [ ] Confirm no duplicate/overlapping keywords cannibalizing

## Step 9: Document
- Update Reporting Log for each client (spend, conversions, CPA, key changes)
- Log budget changes in Budget Shift Log
- Add learnings to client's Agent Memory
- Update Testing Queue

## Step 10: Notify
- Send Slack summary if significant changes made
- Flag anything needing client approval (budget increases, big strategy shifts)
