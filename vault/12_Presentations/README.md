# 12_Presentations

Weekly marketing presentation decks for Dillon OS.

- **Naming convention:** `YYYY-MM-DD Weekly Presentation.md` (date of the Sunday it covers).
- **Generated:** Every Sunday at 5:00 PM from `_Presentation Template.md`.
- **Format:** Marp-compatible Markdown - convert to PPTX/PDF/HTML via the `marp` CLI.
- **Consistency:** Each new deck reflects the prior decks in this folder. Keep the structure identical week over week.
- See `04_SOPs/Sunday Presentation SOP.md` for the full process.
