# Momentum 360 — 60-Second Video Production Package

**Working title:** *Momentum 360: Immersive Content + AI Visibility That Wins*
**Format:** 60.00 sec · 9:16 vertical (1080×1920) · 30fps · **no voiceover**
**Aesthetic:** Matched 1:1 to the supplied reference (`lsahighlight720p.mp4`) — dark "space" backdrop, glowing medallion logo, two‑tone kinetic typography, 3D objects/charts, count‑up data — **re‑skinned to a blue + yellow system** and re‑themed to immersive content + AEO/GEO.

---

## What's in this package

| File | Purpose | Primary user |

## Build order (recommended)
1. Read `01` (concept) and `03` (look) to load the style.
2. Lock the music track per `05` and drop beat markers.
3. Build to `02` scene by scene; pull exact copy from `04`.
4. Layer SFX from `05`; add the logo stinger on the end card.
5. Master 9:16, then reframe to 16:9 / 1:1 per `07`.
6. Cut the thumbnail + end card per `06`.
7. **Fact‑check every on‑screen number against `08` before publishing.**

## Two things to confirm before production (flagged, do not guess)
1. **Exact brand hex + logo file.** The live site (momentumvirtualtours.com) blocked automated color extraction. The blues/yellows in `03` are matched *by eye* to the reference and are production‑ready, but pull the official logo SVG + brand hex from Momentum 360's media kit and reconcile. The reference logo is the circular blue "sky" medallion with the white cursive **m**.
2. **Stat attribution.** All numbers in `04`/`05` are sourced in `08`. Keep the small‑print source line on‑screen with each stat (e.g. "Pew Research, 2025"). Swap any stat you can't stand behind for the safe alternates provided.
