# Conversion Ad Copy Ideas

> Copy templates for conversion-optimized Meta campaigns.

## Short Form (Feed)
### Template
"[Hook - 1 line]
[Benefit 1]
[Benefit 2]
[Benefit 3]
[CTA with urgency]"

### Template
"[Bold claim].
Here's proof: [specific result].
[CTA]."

## Long Form (Feed)
### Template
"[Story hook]
[Problem description - make it relatable]
[How the solution works]
[Specific results / proof]
[Offer details]
[CTA + urgency]"

## Carousel
### Template
Slide 1: Hook / problem
Slide 2: Solution intro
Slide 3: Benefit 1
Slide 4: Benefit 2
Slide 5: CTA / offer

## Video Script (15-30 sec)
### Template
0-3s: Hook (stop the scroll)
3-10s: Problem / pain point
10-20s: Solution + proof
20-30s: CTA + offer

## Client-Specific
### Florecita
- Focus on visuals, minimal copy
- "Your table is waiting" + reservation link

### Buzz Bull
- Results-driven, system-focused
- Lead with ROI or time savings
