# Book Site SEO Sweep

## Trigger
Every Thursday. Checks rank drift, content opportunities, and guest post pipeline for the book site.

## Inputs
- `05_Book/seo-strategy.md` (target keywords, pillar clusters)
- `05_Book/overview.md` (book positioning, author angle)
- `05_Book/guest-post-pipeline.md`
- `05_Book/email-growth-tracker.md`
- Any available rank data (SEMrush, GSC)

## Steps
1. Compare current keyword positions vs prior Thursday snapshot
2. Flag any target keyword with a >5 position drop
3. Scan guest post pipeline for stale outreach (>14 days no reply → flag for nudge)
4. Check email-growth-tracker for trajectory vs 2,000-subscriber / 4-month target
5. Identify 2-3 new blog post opportunities from keyword gaps
6. Write report to `05_Book/seo-reports/YYYY-MM-DD.md`

## Output format
```
# Book SEO Sweep — YYYY-MM-DD

## Rank Movement
| Keyword | Prior | Current | Δ |

## Guest Post Pipeline
• [outlet] — [status] — [days since last touch]

## Email Growth
- Current subscribers: N
- Weekly growth: +N
- Pace vs 2000/4mo target: on track / behind / ahead

## Content Opportunities (This Week)
1. [topic] — [target keyword] — [rationale]

## Flags
• [anything needing attention]
```

## Success criteria
- Report generated by EOD Thursday
- Every >5-position drop has a hypothesis in Flags
- Stale guest-post outreach is surfaced, not buried
