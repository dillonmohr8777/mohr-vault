# Morning Directives

> The drop-box the [[Master Agent]] reads first each morning (Step 0 — Intake). Drop freeform tasks here
> from anywhere — phone note, Slack, a quick line. The commander parses each into a bounded lane,
> classifies its tier, routes it to a skill via `skill-map.json`, and folds it into the same approval
> board. Anything it can't map, it surfaces on the board rather than guessing.

## How to use

- One task per line or bullet. Plain language is fine.
- Tag `#recurring` to keep a directive across runs; otherwise it's cleared after the run.
- Optional hints: name the client, the lane (ads/web/seo/reporting/comms), or a deadline.
